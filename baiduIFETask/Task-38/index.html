任务三十八：UI组件之排序表格

创建了sorTable类用于生成可排序表格。初始化sorTable类创建实例时需要传入一个参数，即用于生成表格的数据，数据格式如下示例：

	var myTable = new sorTable({
		//thead表示将要生成表格的表头
		thead: ["姓名", "语文", "数学", "英语"],
		//tbody表示将要生成表格的主体
		tbody:[
			["小明", "96", "67", "90"],
			["小红", "89", "76", "93"],
			["小刚", "54", "88", "87"],
			["小华","88","99","101"],
			["李雷", "77", "69", "99"],
			["韩梅梅", "87", "77", "43"],
			["吉姆格林","66","43","96"],
		]
	}).appendTo(document.body);

sorTable提供两个接口方法：appendTo(where)用于指定表格添加到哪个DOM元素,支持链式调用，参数where必须是DOM元素。canSort(which, fn)用于指定哪一列需要变成可排序和排序函数fn。示例如下：

	/*第一个参数表示要进行排序的列的列名，是必选参数，
	 *fn是可选参数，如未指定，将按照内置函数排序。
	 *下面的例子指定了“语文”列为可排序列，并提供了排序函数。
	 ************
	 *默认情况下每一列都是不可排序的，需要组件使用者指定。
	 */
	myTable.canSort("语文", function(a, b){
		return b[1] - a[1];
	});
