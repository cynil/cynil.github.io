任务三十七：UI组件之浮出层

POP对象构建浮出层，暴露两个接口方法：alert和confirm。alert方法接收一个参数，即要显示的信息。confirm方法接收两个参数：要显示的信息和点击完按钮后的回调函数，回调函数接收一个参数，表示用户点击的按钮。（没有做浏览器适配，仅在chrome下无bug）

示例：

	if(something happened){
		POP.alert("hello!");
	};

	if(something happened){
		POP.confirm("are you sure?", function(sure){
			if(sure){
				//if the user pressed OK;
				//do something here;
			}
		});
	}

下面是一个测试，当点击清除按钮后，会弹出confirm问你是否清除文字内容，如果你点击sure，就会清除，否则不会，如果已经清除过了，就会显示alert弹窗，提示你已经清除过了。
